# OmniLocal — Marketing Website

Next.js site for OmniLocal, a digital growth agency for local businesses in Kasoa, Ghana.

## Run locally
```
npm install
npm run dev
```
Visit http://localhost:3000

## Deploy
1. Push this project to a new GitHub repository.
2. Go to vercel.com → New Project → Import the GitHub repo.
3. Leave all settings as default (Next.js is auto-detected) → Deploy.

## Edit later
- Colors & fonts: `app/globals.css`
- Copy & sections: `app/page.tsx`
- Contact form logic: `app/ContactForm.tsx`
- WhatsApp number: edit `WA_NUMBER` at the top of `app/page.tsx`
